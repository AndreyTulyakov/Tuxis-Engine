==Tuxis-Engine
Tuxis is a simple, 2D oriented game engine, based on DirectX11 graphics.
Developed in order to study and does not strive for commercial indicators.
It is written in C++, and has bindings only for C++ language.

=== Requirements
* Windows 7 or later
* DirectX SDK
* DirectX11 hardware support
* C++0X Compiler
* Use "AngelCode Bitmap Font Generator" for generate .fnt files.
